The mod object, (aka _mods inside /libisaac.py), is a global dictionary that holds all the information about a mod. It is documented here.

```json
_mods
    ├─ "!!mod config pure"
    │   ├─ path : string        | "/user/path/to/mods/mod_folder"
    │   ├─ index : string       | 0001
    │   ├─ disabled : bool      | False
    │   ╰─ id : int             | (steam id)
    ╰─ "..."
```
